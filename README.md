# Statistics in Action: Distribution Modeling & Non-Parametric Regression

Statistical analysis framework focusing on market "fat tails" and rigorous hypothesis testing on heterogeneous datasets.

##  Objective
Model the daily returns of the S&P500 index accurately and conduct robust statistical inferences.

##  Key Features
- **Distribution Modeling:** Application of Gaussian Mixture Models (Expectation-Maximization) and Location-Scale Student’s t-distributions to capture leptokurtic behaviors.
- **Hypothesis Testing:** Welch's two-sample t-tests and implementation of Benjamini-Hochberg/Bonferroni procedures for FDR/FWER control.
- **Non-Parametric Regression:** Polynomial and exponential regressions to identify non-linear trends.

##  Technologies
`R`, `Statistical Modeling`, `Hypothesis Testing`
